# Image in Code Examples
Every image in a code block should be rendered as literal text, not as an actual image. This applies to both HTML and Markdown image syntax.

Fenced code block with HTML image:

```html
<img src="https://charm.sh/logo.png" alt="Charm Logo">
```

Fenced code block with Markdown image:

```markdown
![Charm Logo](https://charm.sh/logo.png)
```

Indented code block with HTML image:

    <img src="https://example.com/img.png" alt="Logo">

Indented code block with Markdown image:

    ![Logo](https://example.com/img.png)

Inline code with HTML image: `<img src="https://example.com/img.png" alt="Logo">`

Inline code with Markdown image: `![Logo](https://example.com/img.png)`
